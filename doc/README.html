Allgemeines

Monitoring Container zur Auswertung von Lasttest zwischen einer SinaBox und dem IPSEC Lastcontainer.

Aufbau

Der Monitoring Container besteht aus folgenden Komponenten/Containern:

Grafana

Grafana ist ein HTML-Frontend zur Visualisierung der Daten

Influxdb

Influxb ist ein K/V-Store in dem die Messwerte gespeicher werden

Collectdb

Collectd: Sammelt die Daten der BOX per SNMP ein und schreib sie in den K/V-Store

Zum Start des Container bitte immer docker-compose up verwenden, da zwischen den einzelnen Containern Abhängigkeiten bestehenen welche so autom. aufgelöst werden.
